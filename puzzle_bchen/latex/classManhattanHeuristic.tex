\-Manhattan heuristic computation. \-Computes the heuristic of the board by analyzing all the tiles and how far they are from the position they should be. \-For example if the tiles are in proper order in a 3x3 matrix and only value 7 and the blank tile then the algorithm will calculate how far the tile value 7 at index 0 is from its ideal position at index 7 by looking at the horizontal and verticle moves necessary to displace the value 7 tile.
